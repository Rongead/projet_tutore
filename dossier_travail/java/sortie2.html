Générateur de PréAo

1 Objectif

1.1 le programme

l'objectif du programme est à partir d'un unique fichier source de générer un ensemble de pasges au format HTML, qui à tavers un le mode plein-écran d'un navigateur (F11) correspondra à un affichage type projection diaporama

votre programme devra être très facile d'utilisation

la documentation utilisateur devra être suffisante à l'exécution de votre programme

1.2 la soutenance

L'objectif de la soutenance est de nous montrer vos compétences de communication en nous présentant un logiciel quelconcque. vous utiliserez bien évidemment votre outil pour nous présenter votre diaporama
